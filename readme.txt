== Description ==
This is a simple plugin which allows you to add / insert scripts, codes, or texts to the header (head section) and footer (footer section) of your WordPress Website and Blogs.

For Example: Simple copy and paste the Google Analytics code in the text box and save it. 

**Features:**

1. Add / Insert Scripts, Codes, Texts To Head Section (Header).
2. Add / Insert Scripts, Codes, Texts To Footer Section.

== Installation ==
1. Extract all files from the ZIP file, and then upload the plugin's folder to /wp-content/plugins/.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to the plugin settings and enable or disable and customize the options.
